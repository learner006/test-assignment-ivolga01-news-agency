01 PROBLEM
==========

1. PHP и HTML. Напишите код на PHP
В переменной $a лежит текст новости. В переменной $link лежит ссылка на страницу с полным текстом этой новости.
Нужно в переменную $b записать сокращенный текст новости по правилам:
- обрезать до 180 символов
- приписать многоточие
- последние 2 слова и многоточие сделать ссылкой на полный текст новости.
Какие проблемы вы видите в решении этой задачи? Что может пойти не так?
Результат: ссылка на репозиторий с кодом и ваши комментарии.

02 EXPLANATIONS
===============
Проблемы могут идти из трёх источников:
1) плохие входные данные;
2) корректность данных для браузера;
3) корректность данных для PHP engine.


02.1 PHP engine
---------------
Здесь эта проблема вряд ли будет, но не должно быть возможности передать интерпретатору PHP на сервере код для выполнения через функцию exec(), system() например. В php.ini эти функции должны быть отключены disable_functions = system, exec, passthru, shell_exec, proc_open AFAIK

02.2 JavaScript код внутри HTML
-------------------------------
1-е правило при web-разработке: не доверяй никому, даже своей собственной БД (текст новостей хранится в БД). Это означает, что внутри переменной с текстом новости может 
быть JS код, который выполнится в браузере. Этого быть не должно, поэтому будем использовать функцию htmlentities, чтобы "экранировать" угловые скобки и другие "плохие" символы. ( Вообще это PHP best practices и соответствует OWASP :-) ) 

02.3 Входные данных для алгоритма
---------------------------------
Они могут быть различными, но алгоритм должен отработать на них приемлемо.

1) Кодировка символов. Предполагаем Unicode. Это значит, что текст может быть на китайском, арабском языках или иврите.( Ведь мы же создаём лучшее в мире news agency! ;-) )  

2) Входная строка (текст новости) может быть пустой или короче 180 символов. (Хотя проверка строк на пустоту должна быть на триггере в БД, но мы же ведь не доверяем БД.)

3) В строке может не быть пробелов. (Например, длинное географическое название через дефис).
